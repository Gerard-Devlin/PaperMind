## 4.1.0

1. New Feature: personalized arXiv recommendations based on user-defined research keywords.
2. New Feature: relevance-ranked recommendation list with cached arXiv results and manual refresh.
3. New Feature: add recommended arXiv papers to the local library together with their PDFs.
4. New Feature: interactive paper graph for exploring paper, author, topic, and year relationships.
5. New Feature: graph search with Enter-based navigation across multiple matched nodes.
6. New Feature: drag-and-pan graph interaction with collision-aware, elastic node movement.
7. New Feature: export Compare results as PDF with paper-style typography.
8. Improve: downloaded recommendation papers are kept visible and marked in the recommendation table.
9. Improve: recommendation table columns, sorting, publication dates, and library-added state.
10. Improve: Ask evidence interaction can jump back to the corresponding source PDF.
11. Fix: Ctrl/CMD multi-select in sidebar tags/folders no longer depends on the Node `process` global.
